# Flowserve Frontend (React + Vite + TypeScript)

Frontend for the Flowserve challenge. Built with Vite, React, TypeScript, Shadcn-style UI, and Axios.
Configuration is via `.env` -> `VITE_API_BASE_URL` (default `http://localhost:4000`).

## Quick start

1. Install deps:
```bash
npm install
```

2. Create `.env` and set API base URL:
```env
VITE_API_BASE_URL=http://localhost:4000
```

3. Run dev server:
```bash
npm run dev
```

Open http://localhost:5173
