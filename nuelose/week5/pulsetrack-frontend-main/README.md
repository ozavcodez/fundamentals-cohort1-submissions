## Overview

This repository contains the **frontend** for the PulseTrack Health Monitoring Platform.  
It connects to the backend API to display user data, health metrics, and appointment tracking in an intuitive dashboard.

---

## ⚙️ Tech Stack

- **React 18 + TypeScript**
- **Vite** for fast development and builds
- **Tailwind CSS** for modern styling
- **React Router** for navigation
- **ESLint + Prettier** for code consistency

---

## Features

- **Health Dashboard** (real-time data visualization)
- **Meal Tracking**
- **Medical Appointments Management**
- **Responsive Design**
- **Optimized API Integration**

## Getting Started

### Prerequisites

Node.js (>= 18)

npm or yarn

### Installation

**Clone the repo**

```bash

git clone https://github.com/nuelose/pulsetrack-frontend.git
```

### Navigate to the project

```
cd pulsetrack-frontend
```

### Install dependencies

```
npm install
```

### Running the App

```bash

npm run dev
```

The app will start on http://localhost:5173.

### Environment Variables

Create a .env file in the root directory with the following:

```bash
VITE_API_BASE_URL=https://pulsetrack-backend.vercel.app/api
```

### API Integration

Update the base URL in .env to point to your backend server.
